# Event Planner Backend

This is the backend API for the Event Planner project built with Node.js, Express, and MongoDB.

## How to Run

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev` to start the server

## API Endpoints

- GET /api/events
- POST /api/guests
- PUT /api/guests/:id
- DELETE /api/guests/:id
